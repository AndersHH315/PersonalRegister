# PersonalRegister
En enkel console app för att hantera ett personalregister för en restaurang

#Uppgifter

Vilka klasser bör ingå i programmet?
2 klasser, ett för register och ett för personal

Vilka attribut och metoder bör ingå i dessa klasser?
Personal klassen hanterar attributsen för varje person som skrivs in i registret.
Dessa är string Firstname, string Lastname, int Payment och en List<Personnels>

Register klassen hanterar metoderna som används för att lägga till en ny person samt visa listan med all personal.
Dessa metoder är addPerson, createNewPerson och showPersonnels.
addPerson indata parametrar är string Firstname, string Lastname och int Payment vilket createNewPerson tillhandahåller.
createNewPerson och showPersonnels behöver ej några indata parametrar.
